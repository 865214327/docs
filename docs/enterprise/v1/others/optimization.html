提示

  • 兰空图床企业版本是基于 Laravelopen in new window 开发的，Laravel 是一个健壮的、重量级的 Web 开发框架，但是由于它是一个全栈型框架，性能上就成了为人诟病的一个缺点，但是我们可以通过一些优化技巧，优化其加载速度，让网站飞起来。
  • 兰空图床企业版在开发初期，精心设计了数据表结构，开发过程中也针对各种 sql 语句优化。

注意

此教程只针对程序方面的优化，不包含服务器网络带宽方面。

# 常规优化方案

# 优化配置加载

php artisan config:cache

# 优化路由加载

php artisan route:cache

# 优化视图加载

php artisan view:cache

# 进阶优化方案

  • 安装 PHP Opcache 拓展open in new window推荐
  • 使用 Laravel Octaneopen in new window实验性

注意

强烈推荐使用 opcache 拓展，使用 Octane 可能会对使站点产生一些奇怪的副作用。
